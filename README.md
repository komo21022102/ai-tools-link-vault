# Link Vault

一個用來收藏資訊連結的單頁工具網站，可依時間排序與分類管理連結。

## Features

- 新增、編輯、刪除連結
- 依「AI資訊、美食、生活、學習」等分類整理
- 最新優先 / 最舊優先時間排序
- 搜尋標題、網址、分類與備註
- 自動縮圖預覽，並支援手動填入縮圖網址
- 分類新增與管理
- JSON 匯出 / 匯入
- 資料儲存在瀏覽器 localStorage

## Local Preview

直接打開 `index.html`，或用本機伺服器預覽：

```bash
python3 -m http.server 8765
```

然後開啟 `http://localhost:8765/`。
